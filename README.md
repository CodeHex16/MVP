## How to clone the repo with submodules
```bash
git clone --recurse-submodules https://github.com/CodeHex16/MVP.git
```


## How to run the project
```bash
docker compose up -d
```
## How to stop the project
```bash
docker compose down
```

```bash
git pull --recurse-submodules
```
